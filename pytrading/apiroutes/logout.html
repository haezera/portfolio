Route: POST /user/logout
Input: 
{
	sessionId: string
}        

Output:
{
    response: "User logged out!"
}

The /user/logout route logs out a user using their sessionId. The route will return a 400 in the case the sessionId doesn't exist. It will return a 403 if the sessionId does not conform to UUID standards.
