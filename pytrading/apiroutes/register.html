Route: POST /user/register
Input: 
{
    firstName: string 
    lastName: string
    email: string
    password: string
}        

Output:
{
    response: "User successfully registered" 
}

The /user/register route creates a new user on the database, and then returns a userId, such that it can be maintained within a session.
