Route: PUT /user/{userId}/pw/update
Input: 
{
    userName: string 
    oldPassword: string
    newPassword: string
}        

Output:
{
    message: "Password has been updated.
}

The route attempts to update the password of a user. If the oldPassword is incorrect, or the userName is incorrect for the userId, then a status code 400 will be returned.
If the userId does not exist within the database, then a 401 will be returned.
If the userId does not conform with UUID standards, then a 403 will be returned.
