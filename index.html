Projects

trade

trade is a simple full stack project which utilisies the Flask framework, to create a simple way for traders to gather information about tickers on the NYSE and NASDAQ. Currently, the project has finished the development of the API, and the React front-end is currently being built. Check it out on github!

Some key features include:
Backtester
Popular trading strateges (SMA, Awesome Oscillator, VWAP)
Profit/loss indicator in backtest
